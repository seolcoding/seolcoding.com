### 1️⃣ 명확한 의도 전달
AI에게 **무엇을**, **왜**, **어떻게** 만들고 싶은지 구체적으로 설명하세요.
</grid>

<grid drag="48 25" drop="51 1" pad="2em" bg="#F3E5F5" class="fragment">
### 2️⃣ 반복적인 대화
한 번에 완벽한 결과를 기대하지 말고, **대화를 통해** 점진적으로 개선하세요.
</grid>

<grid drag="48 25" drop="1 35" pad="2em" bg="#FFF3E0" class="fragment">
### 3️⃣ 결과 검증
AI가 생성한 코드를 **직접 실행**하고 **테스트**하여 검증하세요.
</grid>

<grid drag="48 25" drop="51 35" pad="2em" bg="#E3F2FD" class="fragment">
### 4️⃣ 학습과 성장
AI의 제안을 **이해**하고, **왜 그렇게 작동하는지** 학습하세요.
</grid>
</grid>

note:

### 자주 묻는 질문 {.fragment}

**Q: API 키가 유출되면 어떻게 하나요?**
A: 즉시 Google Cloud Console에서 키를 삭제하고 새로 발급받으세요.

**Q: 무료 할당량을 초과하면?**
A: Gemini API는 일일 무료 할당량이 있습니다. 초과 시 다음날까지 대기하거나 유료 요금제를 사용하세요.

**Q: 에러가 발생했을 때 어떻게 하나요?**
A: 에러 메시지를 정확히 복사해서 AI에게 물어보세요. 대부분의 에러는 해결 방법이 있습니다.
