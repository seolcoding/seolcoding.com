# 강사 프로필 파일 가이드

이 폴더에는 강사 활동에 활용 가능한 3가지 버전의 프로필이 포함되어 있습니다.

## 📁 파일 구성

### 1. `01_간략형_프로필.md`
**용도**: 빠른 소개, 이메일 서명, SNS 프로필
- 1페이지 분량
- 핵심 경력과 수상 이력만 간단히 정리
- 연락처 및 강의 가능 분야 포함

### 2. `02_표준형_이력서.md`
**용도**: 공식 강사 지원서, 이력서 제출
- 2페이지 분량
- 학력, 경력, 수상, 자격증, 기술 스택 상세 기재
- 강의 가능 주제 및 주요 프로젝트 포함
- 차별화 포인트 강조

### 3. `03_교육특화_프로필.md`
**용도**: 강의 플랫폼 프로필, 교육 기관 제안서
- 교육에 특화된 구성
- 강의 과목 목록 (9개 과정)
- 강의 특징 및 수강생 후기
- 강의 가능 형태 명시

## 🔄 PDF 변환 방법

### 방법 1: Markdown 뷰어 사용
- VS Code에서 Markdown PDF 확장 프로그램 설치
- 파일 열고 `Ctrl+Shift+P` → "Markdown PDF: Export (pdf)" 선택

### 방법 2: 온라인 변환
- https://www.markdowntopdf.com/ 접속
- 파일 업로드 후 다운로드

### 방법 3: Pandoc 사용
```bash
pandoc 01_간략형_프로필.md -o 01_간략형_프로필.pdf
```

## 🎨 커스터마이징 가이드

각 파일은 Markdown 형식으로 작성되어 쉽게 수정 가능합니다:

1. **연락처 업데이트**: 각 파일 상단의 이메일/전화번호 수정
2. **수상 경력 추가**: 수상 섹션에 새로운 항목 추가
3. **강의 과목 추가**: 버전 3의 강의 과목 섹션에 신규 과정 추가
4. **자격증 업데이트**: 새로운 자격증 취득 시 추가

## 💡 활용 팁
