# 전세/월세 계산기 (Rent Calculator)

한국 부동산 임대차 시장의 전세/월세 구조를 이해하고, 합리적인 임대차 계약을 체결할 수 있도록 돕는 웹 기반 계산 도구입니다.

## 주요 기능

### 1. 전세 → 월세 변환
- 전세금과 희망 보증금 입력
- 법정 전월세 전환율 적용 (기본 4.5%)
- 월세 자동 계산
- 계산 공식 및 결과 상세 표시

### 2. 월세 → 전세 역계산
- 월세 보증금과 월세 입력
- 전세 환산액 자동 계산
- 현재 월세와 환산 전세 비교

### 3. 비용 비교 분석 (2년 기준)
- 전세 vs 월세 총 비용 비교
- 중개수수료 자동 계산
- 대출 이자 포함 계산
- 손익분기점 분석
- 시각화 차트 (Recharts)

## 기술 스택

- **React 19**: UI 라이브러리
- **TypeScript**: 타입 안전성
- **Zustand**: 상태 관리

## 계산 로직

### 전세 → 월세
```
월세 = (전세금 - 보증금) × 전환율 ÷ 12
```

### 월세 → 전세
```
전세 환산액 = 보증금 + (월세 × 12 ÷ 전환율)
```

### 중개수수료
- 주택임대차보호법 시행규칙 제20조 기준
- 거래금액별 상한 요율 적용
- 월세는 환산 거래금액으로 계산

### 손익분기점
```
손익분기점 = (전세 초기비용 - 월세 초기비용) ÷ (월세 월부담액 - 전세 월부담액)
```

## 법적 근거

- 주택임대차보호법 제7조의2 (월차임 전환 시 산정률의 제한)
- 주택임대차보호법 시행령 제9조 (월차임 전환 시 산정률)
- 공인중개사법 시행규칙 제20조 (중개보수 등)
- 법정 전환율 상한: MIN(10%, 기준금리 + 2%)

## 주의사항

본 계산기는 참고용 도구이며, 법적 구속력이 없습니다.
실제 계약 시에는 반드시 전문가(공인중개사, 변호사)와 상담하시기 바랍니다.
